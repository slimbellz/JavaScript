JavaScript Arrays

Creating an Array

var cars = ["Saab", "Volvo", "BMW"];

document.getElementById("id").innerHTML = cars;

Changing an element in an array

var ncars = ["Saab", "Volvo", "BMW"];

ncars[0] = "Opel";

document.getElementById("id").innerHTML = ncars;

Adding Array Elements

fruits.push("Lemon");// adds a new element (Lemon) to fruits

Converting Array to String

The toString() method returns an array as a comma separated string

document.getElementById("id").innerHTML = fruits.toString();

Popping

The pop() method removes the last element from an array:

fruits.pop();//removes the last element Grape.

Pushing

The push() method returns the new array length:

fruits.push("Kiwi");

Shifting Elements

Similar to popping the difference is it works on the first element

fruits.shift();

unshifting Elements

Similar to pushing the difference is it adds new element to the beginning of the array

fruits.unshift("Coconut");

Splicing an Array

concat

The concat() method is used to merge (concatenate) arrays:

let concat1 = ["Mohammed", "Kafayat"];

let concat2 = ["Tahira", "Temitope", "Olatundun"];

let concat3 = ["Olalekan", "Mopelola", "Bellos"];

let myFamily = concat1.concat(concat2, concat3);

document.getElementById("id").innerHTML = myFamily;
