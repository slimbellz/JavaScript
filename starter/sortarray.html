Arrays Continuation

Sorting

The sort() method sorts an array alphabetically.

var fruits = ["Banana", "Orange", "Apple", "Mango", "Cashew", "Lemon", "Sugarcane"];

document.getElementById("sortit").innerHTML = fruits;

function myFunction() {

fruits.sort();// Sorts the elements of fruits

document.getElementById("sortit").innerHTML = fruits;

}

Reversing an Array

The reverse() method reverses the element in an array.

You can use it to sort array in descending order:

fruits.sort();//First sort the elements of the fruits

fruits.reverse();//Then reverse the order of the elements
